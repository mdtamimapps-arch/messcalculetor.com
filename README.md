# 🍛 The_Octa_Squad – Mess Management System (Bangla Cash Memo)

A complete web-based **Mess Management System** designed for 8-member mess/co-living groups.  
Easily record daily bazaar (grocery) expenses, get Bengali product suggestions, auto-calculate totals, and print/download as PDF.

## ✨ Features

- ✅ **Smart Bengali Product Suggestions**  
  Type the first letter in Bangla (e.g., "স" for সবজি) – system recommends items instantly.

- 📋 **Itemised Cash Memo Table**  
  Columns: Serial, Product Name, Quantity, Unit Price, Total, Actions (Edit/Delete).

- 🧮 **Automatic Grand Total**  
  Updates in real-time as you add/edit items.

- 🗓 **Auto Date & Time**  
  Latest date and time shown in the header – updates every minute.

- 👥 **Manager & "Buy By" Fields**  
  Store who managed the mess and who purchased the items (pre-filled with 8 member names).

- 🖨 **Print / PDF Export**  
  Optimised print styles – perfect for printing as a physical cash memo or saving as PDF.

- 🧹 **Reset All** – Clear all items with confirmation.

- ✏️ **Edit & Delete** – Modify any existing entry easily.

## 🖥️ Live Demo

You can directly open the `index.html` file in any modern browser – no server required.

## 🚀 How to Use

1. **Add items**  
   - Start typing a product name (Bengali or English).  
   - Suggestions appear below the input field.  
   - Enter quantity (kg/piece) and price per unit.  
   - Click "আইটেম যোগ করুন".

2. **Edit / Delete**  
   - Click the ✏️ edit icon to modify an item.  
   - Click the 🗑️ delete icon to remove an item.

3. **Set Manager & Buyer**  
   - Fill in the Manager name.  
   - Choose or type the buyer name (8-member list provided).

4. **Print / PDF**  
   - Click the **PDF / প্রিন্ট করুন** button.  
   - Use browser's print dialog (Ctrl+P) → Save as PDF or Print.

5. **Reset** – Use the "সব মুছুন" button to clear the entire list.
